How to Navigate This Application

Use the navigation tabs at the top of the page or the floating tab buttons on the right side of your screen to switch between different sections. On mobile devices, use the bottom navigation bar.

Each tab provides access to specialized tools and reports designed for executive and management performance monitoring.

Admin

Access the administrative control panel for managing system settings, user permissions, and configuration options. This section is intended for authorized administrators only.

Voided

Review and manage voided transactions. This section displays all cancelled or reversed entries, providing a complete audit trail for financial reconciliation and compliance purposes.

Attachments

View and manage all document attachments associated with transactions and records. Upload, download, and organize supporting documentation for easy reference and compliance.

Quick Tips

  • Use the theme toggle in the header to switch between dark and light modes
  • The floating buttons on the right provide quick access to all sections
  • On mobile, swipe or use the bottom navigation bar to switch tabs
  • Use the chatbot (green button) for assistance with common questions
